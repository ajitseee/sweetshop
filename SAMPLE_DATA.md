## Usage Instructions

1. Start your application (both frontend and backend)
2. Register a new user account
3. Make yourself an admin in MongoDB Atlas
4. Log back in as admin
5. Use the "Add New Sweet" button
6. Copy and paste the details from above
7. Add as many sweets as you want to test the application!
